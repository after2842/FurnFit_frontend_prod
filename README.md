# Play

This is the frontend for the Play application (temp name I randomly did), built with [Next.js](https://nextjs.org).

## Project Overview

This project is developed for **CS 125**. We are building a software product that provides **personalized furniture recommendations** based on specific user context, such as room dimensions, style preferences, and lifestyle needs.

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Contains the routes and main page logic.
- `components/`: Reusable UI components.
- `lib/`: Utility functions and shared logic.
